Rawa Elok Smartcity

Smart Kampung adalah aplikasi berbasis web dan insya allah ada versi android nya jg, aplikasi ini bertujuan untuk mempermudah mendata warga yang ada disuatu kampung, meliputi data keluarga , data diri dari data keluarga , alamat rumah , pertanyaan dari masyarakat, request untuk mengurus keperluan dengan RT setempat dan ada berita seputar info yang perlu di sosialisasikan kepada warga sekitar, jadi aplikasi ini mendukung jg program pemerintah dimana dimasa pandemi covid-19 ini kegiatan sosial secara live agak dibatasi dan dianjurkan menggunakan internet atau media sosial seperti aplikasi ini
